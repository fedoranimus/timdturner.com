Synapse
Relaxation Neurofeedback Game
Overview

Synapse is mobile game to help users develop a daily relaxation routine and provide neural-feedback through EEG devices such as the Emotiv Insight.

Due to a lack of Mobile SDKs for consumer EEG devices, Synapse is currently in the wireframing stages.

Prototype

You can click through a wireframe prototype.

The ideas is that the user would use an EEG Headset to measure their relaxation. There is a daily meditation quest available to the user which grants them "Charge". Charge can be used to destroy enemies and spawn additionals synapses, which make the user more powerful.

Going Forward

I hope to eventually complete and release this game. I feel it will provide access and information to people about the benefits of neurofeedback and meditation.
